## __Linux Installation Guide and Usage__

```
 sudo apt update
```
```
 sudo apt install git
```
```
 git clone https://github.com/mkdirlove/pisowifi-ssh-bruter
```
```
 cd pisowifi-ssh-bruter
```
```
 chmod +x *
```
```
 bash install.sh
```
```
 bash pisowifi-ssh-bruter.sh
```
